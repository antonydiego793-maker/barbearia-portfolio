# 🍔 Hamburgueria — Projeto de Portfólio

Site fictício de uma hamburgueria desenvolvido como projeto pessoal para portfólio, com foco em praticar HTML, CSS e JavaScript na criação de uma interface moderna e responsiva.

---

## 📋 Sobre o Projeto

O site simula a presença digital de uma hamburgueria, contando com páginas de apresentação, cardápio e contato. O objetivo foi desenvolver uma experiência agradável ao usuário utilizando apenas tecnologias web fundamentais.

---

## ✨ Funcionalidades

- 🏠 **Página inicial** — apresentação visual da hamburgueria
- 🍔 **Menu** — exibição dos itens disponíveis com descrição
- 📞 **Contato** — seção para o cliente entrar em contato
